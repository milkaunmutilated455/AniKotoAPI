AniKatoAPI

RESTful API for Anime Data from anikototv.to

Node.js Express Cheerio Vercel

Base URL

https://anikatoapi.vercel.app/api

API Endpoints

GET /api/

Get home page data including spotlight, trending, top anime, etc.

GET /api/info?id={anime-slug}

Get detailed information about a specific anime

GET /api/search?keyword={keyword}&page={page}

Search for anime by keyword

GET /api/search/suggest?keyword={keyword}

Get search suggestions

GET /api/episodes/{anime-slug}

Get episode list for an anime

GET /api/stream?id={episode-id}

Get streaming servers for an episode

GET /api/schedule?date={YYYY-MM-DD}

Get anime schedule for a specific date

GET /api/spotlight

Get spotlight/featured anime

GET /api/trending

Get trending anime

GET /api/top-ten

Get top 10 anime (today, week, month)

GET /api/suggestions?keyword={keyword}

Get anime suggestions

GET /api/random

Get random anime information

GET /api/most-popular?page={page}

Get most popular anime

GET /api/filter?{params}

Filter anime by genre, type, status, etc.

GET /api/{category}?page={page}

Get anime by category (genre/action, type/tv, etc.)

URL Structure

Anime: /watch/{anime-slug}
Episode: /watch/{anime-slug}/ep-{number}
Search: /filter?keyword={query}
Genre: /genre/{genre-slug}
Type: /type/{type-slug}
Status: /status/{status-slug}
