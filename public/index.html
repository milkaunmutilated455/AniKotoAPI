Features

Built for developers

Everything you need to build an anime website, mobile app, or Discord bot. No API keys, no rate limits, no hassle.

Full-text Search
Search by keyword, genre, type, status. Get autocomplete suggestions and paginated results.
Episode Streaming
Retrieve streaming servers, source links, and m3u8 URLs for any episode via AJAX endpoints.
Rich Metadata
Synopsis, MAL score, genres, characters, relations, recommendations, and more.
Schedule & Calendar
View what's airing today or fetch the full weekly broadcast schedule.
Advanced Filters
Filter by genre, type, status, season, sort order. Multi-select support.
Fast & Cached
In-memory cache with 5-minute TTL for snappy responses and reduced scraping overhead.
